# Design System: 27-react-pages-demo

## Visual Philosophy
The design follows a **Modern Editorial / Page Hub** aesthetic. It uses a "Clean Paper" foundation with "Editorial Monochrome" accents, focusing on whitespace, hierarchical typography, and modular balance.

## Design Patterns
-   **Cover Hero**: Massive, high-precision typography (9rem) with modular subtitle overlays and kinetic CTA elements.
-   **Modular Hub**: Fixed navigation (H-24) with high backdrop blur (20px) and minimal internal borders.
-   **Grid Anthologies**: Case study containers with 1px black borders (5% opacity) and expressive scale-motion transitions.
-   **Editorial Stroke**: Monospaced metadata (10px uppercase) with high tracking (0.5em) for a curator feel.

## Color Palette
-   **Paper White**: Foundation Background (White)
-   **Monochrome Black**: Primary Text & Graphics (Slate-950)
-   **Editorial Slate**: Auxiliary Text (Slate-400)
-   **Link Accent**: Interactive Highlight (Custom)

## Interaction Design
-   **Hero Entrance**: 1000ms duration translate-fade animations for the main typographical headings.
-   **Collective Pulse**: Subtle background blur-glow on the newsletter section to signify inclusion.
-   **Pattern Kinetics**: 700ms scaling and reveal transitions for case study cards to improve browsing depth.
